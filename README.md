# 💼 Personal Portfolio Website

**Project 1 — Internship Submission**
A responsive single-page portfolio website built with pure HTML and CSS.

---

## 📌 Overview

This is a clean, modern single-page portfolio website built using **HTML and CSS only** — no frameworks or JavaScript libraries. It showcases personal information, skills, projects, and contact details in a structured and visually appealing layout.

---

## 🚀 Live Demo

> Open `index.html` in any browser to view the site locally.

---

## 📁 Project Structure

```
Syntecxhub_Portfolio/
│
└── index.html       # Single HTML file (contains all HTML, CSS & JS)
```

---

## ✅ Features

- **Single-page layout** with smooth scroll navigation
- **4 Core Sections:** About, Skills, Projects, and Contact
- **Responsive design** — works on mobile, tablet, and desktop
- **Hover effects** on nav links, skill cards, project cards, and buttons
- **CSS transitions** for smooth interactive feedback
- **Mobile hamburger menu** for small screens
- **Modern dark color scheme** with mint-green accent
- **Contact form** with submit feedback animation

---

## 🎨 Design Choices

## 🗂️ Sections Breakdown

### 1. Hero

- Name and title
- Short tagline
- CTA buttons linking to Projects and Contact

### 2. About

- Personal bio
- Key stats (projects, years coding, technologies)

### 3. Skills

- 8 skill cards with progress bars
- Hover lift effect with accent border glow

### 4. Projects

- 3 project cards with thumbnail, tags, description, and links

- Contact info (email, LinkedIn, GitHub)
- Contact form with name, email, and message fields

## 🛠️ How to Run

1. Download or clone the project
2. Open `portfolio.html` in any modern browser
3. No build tools or installations required

---

## 📝 Customization Guide

To personalize this portfolio, update the following in `portfolio.html`:

- **Name & tagline** → Hero section (`<h1>` and subtitle paragraph)
- **Bio & stats** → About section
- **Skills** → Skill cards (name + bar width percentage)
- **Projects** → Project cards (name, description, tags, links)
- **Contact info** → Email, LinkedIn, GitHub links
- **Color scheme** → CSS variables in `:root` at the top of `<style>`
